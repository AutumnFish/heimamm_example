1. 下载示例代码，装包并运行
2. 后台接口记得配置起来,[传送门](https://www.yuque.com/prqozm/roh46t/dmadc4)
3. 接口设置了跨域限制,打开浏览器的时候注意自己的`url`地址
   1. 只允许：`http://localhost:8080`端口
   2. 自己配置允许跨域的`chrome` [传送门](https://www.cnblogs.com/linsx/p/9777954.html)
